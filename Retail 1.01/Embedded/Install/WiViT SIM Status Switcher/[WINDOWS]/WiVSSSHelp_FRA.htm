Introduction

A Propos de SSS

SIM Status Switcher vous offre facilement la facilité de contrôler l'état de vos SIM depuis la page Aujourd'hui.

Caractéristiques:

  • Si vous avez un adaptateur pour 2 SIM en place, vous avez la possibilité de basculer facilement de l'une sur l'autre depuis la page Aujourd'hui, sans avoir besoin d'éteindre puis de rallumer le téléphone ou entrer le code PIN.
  • La possibilité d'allumer ou d'éteindre votre GSM Radio sans impact sur le Bluetooth ou le WiFi.
  • Toutes les informations et états affichées sur la page Aujourd'hui sont configurables.
  • Le support des différentes langues est possible par édition en mode texte du fichier WiVSSS.lang
  • Utilisation minimale de la mémoire pour la page Aujourd'hui.
  • Support complet de WM2003SE incluant le mode lanscape et l'usage sans stylet.

Configuration nécessaire:

  • Windows Mobile 2003 Phone Edition.
  • Dual SIM Adaptateur (mais peut être utilisé sans).

Ecran Plugin

Une fois que le plugin est actif, la page Aujourd'hui affiche ce qui suit:

Prêt: [numéro de téléphone]
Opérateur: [Fournisseur du service] ([National ou Etendu])

Les différentes couleurs d'icônes qui s'affichent ont la signification suivante:

  • Blanc (L'état est actuellement disponible)
  • Rouge (Le téléphone est éteint ou la SIM non prête)
  • Ambre (Le téléphone est allumé, la SIM est prête, mais la liste des contacts n'est pas encore lue)
  • Vert (Le téléphone est allumé, la SIM est prête et la liste des contacts est prête à être utilisée)

Les actions d’appui ou maintien sont disponibles après l'installation mais peuvent êtres changées depuis les options du PlugIn:

  • Simple appui (Actualiser les informations du Plugin)
  • Appui avec maintien (Montrer le menu Pop Up)
  • Appui sur icône (Allume ou Eteint le téléphone)
  • Appui et maintien de l'icône (Bascule sur la SIM non active)

Menu Popup

Les menus suivants sont disponibles:

  • Options (Permet de paramétrer SIM Status Switcher)
  • Actualiser (Actualise les données du Plugin)
  • Changer de SIM (Bascule sur autre de SIM)
  • Allumer/Eteindre (Allume/Eteint uniquement le GSM. Bluetooth ou Wifi ne sont pas affectés)
  • Réglage du Téléphone (Règle vos paramètres du PlugIn)

Options du PlugIn

L'état des options du PlugIn peuvent être affiché en utilisant une des méthodes suivantes:

  • Par utilisation des actions Appui ou Appui et Maintien sur le PlugIn dans la page Aujourd'hui.

    Par défaut, l'action utilisée pour afficher les options est depuis le menu PopUp en tapant avec ou sans maintien sur la PlugIn.
    Les actions peuvent êtres configurées en utilisant le sous menu « Display ».
  • Par appel des options du PlugIn depuis le réglage de la page Aujourd'hui depuis la Configuration des paramètres.

    Noter que les options ne sont disponibles que si le PlugIn est actif

Depuis les options du PlugIn, les pages suivantes sont disponibles:

  • Affichage
  • Aspect
  • Actions
  • Securité
  • Langue
  • A Propos

Quand vous faite une modification, validez par l'icône OK pour sauver les nouveaux paramètres et sortir des options. Les changements sont effectifs immédiatement au retour sur la page Aujourd'hui.

Si vous voulez détruire les changements réalisés, appuyez sur l'icône X. Si les changements sont effectifs, vous serez interrogé par un message pour être sûr que vous annulez les changements. Appuyez sur Oui pour sortir sans sauvegarder les changements. Si vous ne voulez pas annuler, confirmer par le bouton NON.

Aussi depuis n'importe quelle fenêtre des options du PlugIn, il existe un lien en bas d'écran vous permettant d'accéder à vos paramètres de téléphone.

Réglage d'Affichage

La fenêtre « Affichage » vous permet de régler la façon dont les informations sont affichées via des cases à cocher Les combinaisons suivantes sont possibles:

  • Afficher le numéro et le fournisseur sur une seule ligne.
  • Afficher le numéro sur une seule ligne.
  • Afficher le fournisseur sur la ligne deux.

L'état de la SIM (Marche/Arrêt) est valide sur toutes les combinaisons.

Réglages d'aspect

La fenêtre « Aspect » vous permet de modifier les informations visibles sur le PlugIn. Les changements possibles sont:

  • Passer en gras la ligne 1 et/ou la ligne 2.
  • Changer le type d'icône utilisée sur le PlugIn.
    Les icônes pré-définies sont les suivantes:
    • Téléphone Standard
    • Boutons Combiné
    • Téléphone mobile
    • PDA
    • Feux Tricolores
    • Lampes colorées

Réglages des actions

La fenêtre « Actions » vous permet de changer quelle action à réaliser sur simple appui ou Appui et maintien. Les actions suivantes peuvent êtres paramétrées:

  • Simple appui
  • Appui et maintien
  • Simple appui sur Icône
  • Appui et maintien sur icône

Chaque action peut être assignée comme suit:

  • Actualise (Applicable au PlugIn)
  • Monte Menu PopUp (accéder au menu PopUp)
  • Options (Accéder aux options de SIM Status Switcher)
  • Change de SIM (commute de SIM)
  • Allumer/Eteindre le Tél (Ne concerne que le GSM)
  • Réglages Téléphone (Accès direct aux options du Téléphone)

Les actions par défaut après installation sont:

  • Appui simple: Actualise
  • Appui et Maintien: Montre Menu PopUp
  • Simple appui sur Icone: Allumer/Eteindre le Tél
  • Appui et maintien sur Icone: Change de SIM

Réglage de sécurité

La fenêtre « Securité » vous permet de rentrer automatiquement le code PIN de carte SIM actuellement en fonction. Deux niveaux de code PIN sont disponibles:

  • Après un Soft reset ou validation du PlugIn
  • Quand le GSM est allumé

Vous pouvez sélecter une ou deux cases à cocher. Après sélection, appuyez sur le bouton [Création PIN], entrer votre Code PIN et valider par OK. Vous devrez alors le ré-entrer pour confirmer votre entré correcte. Appuyer sur OK pour finir. L'appui sur Annulez à n'importe quel moment annule l'opération courante.

Si vous voulez changer le Code PIN, vous devez le faire par le bouton [Modifier PIN]. Vous devrez d'abord confirmer le code actif, après quoi il faudra entrer le nouveau code. Ré-entrer ensuite le code pour confirmer votre entré correcte. Appuyer sur OK pour finir. L'appui sur Annulez à n'importe quel moment annule l'opération courante.

Pour régler la même chose sur votre deuxième SIM, basculer d'abord sur celle-ci, puis effectuer ensuite les opérations ci-dessus.

Le code PIN peut être différent pour chaque SIM.

Réglage de Langue

La fenêtre "Langue" vous donne la possibilité de changer la langue utilisée dans les différents messages d'information.
La langue courante est affichée parmis les différentes qui sont disponibles

Fenêtre A Propos

Cette fenêtre vous montre les informations de version et copyright relative au produit. Elle vous permet aussi d'afficher l'état de la licence (Test, Bêta ou Enregistrée).

Depuis cette fenêtre vous accédez aussi aux dialogues suivants:

  • Licence
  • Information

Fenêtre Information

La fenêtre « Information » sous renseigne sur:

  • Information du PDA (Constructeur, Modèle, Révision, numéro de série)
  • Information de SIM (Numéro téléphone, Identifiant, numéro de série)
  • Stockage des contacts (Emplacement, Nombre d'entrés, Entré utilisée)

Dialogue d'Enregistrement

Entrez le code de 20 caractères fourni par WiViT dans les 5 boites affichées.

Une fois entré le code de licence, validez par appui sur OK pour sauvegarder et retourner sur la fenêtre « A Propos ».
Pour que la licence soit activée, vous devez soit faire un soft reset, soit désactiver et réactiver le PlugIn depuis le paramètrage de la page Aujourd'hui dans la fenêtre des paramètres du PDA.

Après ceci la fenêtre « About » vous informe de l'état de la licence entrée.
